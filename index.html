What are the basics of HTML?

Before We get Into That, You must know the Full Form of HTML is Hyper Text Markup Language. Remember That This is NOT a Programming Language. This is Called A Markup Language Because it Defines The Layout Of the Page, So that The Browser Can display it For You. HTML is NOT Case Sensitive, Meaning that it doesn't need a capital or small letter.

The <ol> and <ul> Tags

The ol and ul tags are Ordered List and Un-Ordered Lists Respectively. They are important when you want to make a list in your website. You might wonder about the difference between The list tags. The ol tag gives you Numbers, Letters, Etc. However, The ul tag gives you bullet points which are useful when there are many things, They reduce the usage of numbers. There is a common Misconception where people think that you have to apply a number before adding the li tag, Which is the one where you Make a point of a list, Like <li> Write your stuff here and close the tag using </li>. So this is the way of making a list item. Now, Coming back to the topic, The Text Editor will automatically add the numbers and the bullet points for you, Making your work easy.
