What are the basics of HTML?

Before We get Into That, You must know the Full Form of HTML is Hyper Text Markup Language. Remember That This is NOT a Programming Language. This is Called A Markup Language Because it Defines The Layout Of the Page, So that The Browser Can display it For You.

Starting Off:

There are a few Basic Commands That you need to learn about HTML. The Version Of HTML we Will be Using is HTML 5. This is the Latest Version Yet.

The "<head>" Tag

The Head Tag Is the part of your website which is Visible just to you. It consists of the <Title> Tag, Which is the Name of your Website. This is displayed at the tab of your website, Which Is Above the URL Bar.

If you make a Website, You should Experiment with the <Title> Tag as it gives you a better understanding of the use of the tag.
